# AI-call-center-System-demo
This is a full working DEMO architecture of an AI-powered call center that can:  Receive phone calls  Talk to customers using AI  Handle multiple callers simultaneously  Run 24/7 on the cloud  ⚠️ Important: This is a production-style demo, not Twilio-verified live calling yet. Once you add real API keys and phone numbers, it becomes live.

 AI Call Center with AI Agents

 Features
- AI-powered phone agents
- 24/7 availability
- Handles multiple calls
- Easy to scale

 Setup
1. Install Node.js & Python
2. Start Python backend
3. Start Node server
4. Configure Twilio webhook to `/voice`
